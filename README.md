# Ranti

Ranti est le cahier de loyers moderne des propriétaires africains.

## Statut

Reconstruction propre depuis zéro.

Ce dépôt est la source de vérité produit et technique de Ranti.

## Problème unique

Aider un propriétaire à savoir clairement :

1. qui a payé ;
2. qui est en retard ;
3. quelle preuve existe pour chaque paiement.

## Règle de construction

Aucune fonctionnalité n'entre dans Ranti si elle ne rend pas plus simple le fait de savoir qui a payé, qui doit payer, ou de prouver qu'un paiement a été effectué.
